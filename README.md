# Challenge-Encriptador
Desafio 1 Alura One 

Desafío número 1 del curso Alura One

En este deafio se nos pidio reemplazar:
la "a" con "ai"
la "e" con "enter"
la "i" con "imes"
la "o" con "ober"
la "u" con "ufat"

Y que no solo se pueda ingresar texto en minúsculas, sin acentos o caracteres especiales.
